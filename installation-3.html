Install and configure Tor

Install Tor:

sudo apt-get install tor

Configure hidden service: 

cd /etc/tor/
sudo cp torrc torrc.default
wget https://raw.githubusercontent.com/GUSecLab/privDoorbell/master/RPI/conf/torrc

Restart Tor:

sudo service tor restart
