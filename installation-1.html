Install the dependencies

Python

The server is based on Python/Flask. Install Python3 and pip if not yet installed: 
sudo apt-get install python3
sudo apt-get install python3-pip

And the Python packages:

sudo pip3 install firebase-admin==4.3.0 Flask==1.0.2 gpiozero==1.5.1 imutils==0.5.3 Jinja2==2.10 numpy==1.16.2 protobuf==3.12.2 pycryptodomex==3.9.7

It is strongly recommended that you install the exactly same version.

OpenCV

The project needs OpenCV and its python wrapper. Installing opencv-python is a bit tricky on Raspberry Pi. Be aware of never running pip3 install opencv-python or pip3 install opencv-python --upgrade. The newest version of opencv-python does NOT work on Raspberry Pi. 

sudo apt-get install python3-opencv
sudo apt-get install libhdf5-dev
sudo apt-get install libhdf5-serial-dev
sudo apt-get install libatlas-base-dev
sudo apt-get install libjasper-dev 
sudo apt-get install libqtgui4 
sudo apt-get install libqt4-test

FFmpeg 

sudo apt-get install ffmpeg
